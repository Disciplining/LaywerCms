#### <font color="blue">获得所有轮播图片的数据</font>
- 请求地址：`/selectData/getCarouselData`
- 请求方式：`GET` ajax提交
- 返回数据类型：`json`
- 返回数据示例
    ```json
    {
        "code": 0,
        "msg": null,
        "count": 2,
        "data": 
        [
            {
                "id": 1,
                "bannerName": "banner1",
                "imgPath": "/images/banner-pic/04.jpg",
                "editDate": "2019-01-01"
            },
            {
                "id": 2,
                "bannerName": "banner2",
                "imgPath": "http://xxx",
                "editDate": "2018-08-09"
            }
        ]
    }
    ```

#### <font color="blue">添加轮播图片</font>
- 请求地址：`/insertData/insertOneCarousel`
- 请求方式：`POST` 表单提交
- 字段名称
    ```
    bannerName
    file
    ```
